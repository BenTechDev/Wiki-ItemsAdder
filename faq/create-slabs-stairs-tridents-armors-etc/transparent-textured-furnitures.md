# רהיטים שקופים

{% hint style="warning" %}
אם אתם רוצים לעשות כמה טקסטרות שקופות או חצי שקופים, אתם צריך לוודא שהטקסטורה מבוססת על פריט וניל שמקבל שקיפות.
{% endhint %}

### לדוגמא:

```yaml
resource:
  material: WHITE_STAINED_GLASS
  generate: false
  model_path: item/your_model
```
